# ER / KG4ER

本目录只保留 KG4ER 试题推荐实验代码。

KG4ER 将学生掌握度（mastery）、学习进展（sequence/progress）和遗忘信息（forgetting）表示为教育知识图谱中的关系，通过知识图谱嵌入完成候选试题排序，并使用 ACC、NOV 等指标评估推荐结果。

主要代码位于：

```text
KG4ER/codes/   模型训练、预测、消融、基线和评估
KG4ER/data/    数据预处理和实验前置文件生成
KG4ER/tests/   自动化测试
```

论文修改材料、审稿回复文档、数据集和实验运行结果均不在 GitHub 中保存。
